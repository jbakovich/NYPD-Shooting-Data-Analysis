# NYPD-Shooting-Data-Analysis
DTSA 5301 Final Project

To download this project for yourself, you can do one of two things: 

1) Enter the NypdShootingDataProject.Rmd file. At the top right of the file, click "Download Raw File".
This will download the file. Now open the file using RStudio, and knit the file as you wish to get a produced solution.

2) Clone the git repository locally and open the .Rmd in your cloned directory
